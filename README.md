## Description

Live application is deployed on Heroku.

## Usage 

# Run the app locally
  1. Open console
  1. Navigate to project root directory
  1. RUn "npm install" to install project dependencies
  1. Run "npm start"
  1. Open browser and type address "http://localhost:3002/"
